  • manage.py

    Это основной файл, который используется для управления проектом Django. Он позволяет выполнять различные задачи, такие как запуск сервера разработки, создание миграции и т.д.

  • settings.py

    Содержит настройки проекта, такие как параметры базы данных, секретные ключи и другие параметры конфигурации.

  • urls.py

    Файл URL-адресов, содержащий список URL-маршрутов для вашего проекта.

  • wsgi.py

    Модуль WSGI, используемый для настройки веб-сервера перед обработкой запросов.

  • templates/

    Папка с HTML-шаблонами для представления данных.

  • static/

    Папка для хранения статических файлов, таких как CSS, JavaScript и изображения.

  • apps.py

    Корневой файл приложения, содержащий информацию о приложении и его моделях.

  • models.py

    Файл моделей, содержащий определения моделей Django.

  • tests.py

    Содержит тестовые сценарии для моделей и представлений.

  • views.py

    Содержит определения функций представлений Django.
